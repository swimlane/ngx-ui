Navbar

Navbar with sliding active indicator. Use swim-navbar with swim-navbar-item children. Set bar-at-top to show the bar above items; use active and active-change for two-way binding.

Active bar on bottom

<swim-navbar>
  <swim-navbar-item><swim-icon font-icon="apps"></swim-icon></swim-navbar-item>
  <swim-navbar-item><swim-icon font-icon="chart-scatter"></swim-icon></swim-navbar-item>
  <swim-navbar-item><swim-icon font-icon="database"></swim-icon></swim-navbar-item>
  <swim-navbar-item><swim-icon font-icon="commandline"></swim-icon></swim-navbar-item>
</swim-navbar>

Active bar on top

Use goTo(index) to activate an item programmatically.

Activate 4th nav item 
<swim-navbar id="navbarTop" bar-at-top>
  <swim-navbar-item><swim-icon font-icon="apps"></swim-icon></swim-navbar-item>
  ...
</swim-navbar>
<swim-button id="btn">Activate 4th nav item</swim-button>
<script>
  document.getElementById('btn').addEventListener('click', () => {
    document.getElementById('navbarTop').goTo(3);
  });
</script>

Usage

swim-navbar: active (number), bar-at-top (boolean). Fires active-change (detail: number). Method goTo(index).

swim-navbar-item: Content in default slot (e.g. icon). Index/total/active are set by the parent navbar.
