Progress Spinner

Circular progress indicator matching ngx-progress-spinner: determinate or indeterminate mode, optional center icons (upload, success, failure), and optional labels.

Change the options to update the spinner. Controls on the left; live preview and code on the right. 

<swim-progress-spinner
  mode="indeterminate"
  value="35"
  total="100"
  diameter="100"
  stroke-width="5"
  color="lime"
  appearance="icon"
  aria-label="..."
></swim-progress-spinner>

Default and determinate modes. Use value and total for progress; when value >= total, the spinner shows complete (and optional icon).

Indeterminate (default)
Determinate 50%
Complete (value = total)

appearance="icon" shows center swim-icons: cloud-upload while in progress, check on success, x on failure (or use slots/icon-name props to override).

In progress (icon)
Complete success
Complete failure

Pass icon names to render swim-icon in the center, or slot custom content (e.g. <swim-icon slot="in-progress-icon" font-icon="upload">). Import @swimlane/lit-ui/icon when using icon names.

In progress (icon name)
Complete (icon name)
Slotted swim-icon

Diameter and stroke-width control size and thickness.

Small (diameter=50)
Default (100)
Large (150)

spinner-label object provides inProgressLabel, completeLabel, and failLabel (set via property in script).

Label text is set via spinnerLabel property in demo (see main.ts).

Import and use the progress spinner component:

<script type="module">
  import '@swimlane/lit-ui/progress-spinner';
  import '@swimlane/lit-ui/icon';
  import '@swimlane/lit-ui/section';
</script>

<swim-progress-spinner mode="indeterminate" aria-label="Loading"></swim-progress-spinner>
<swim-progress-spinner mode="determinate" value="50" total="100" aria-label="Progress"></swim-progress-spinner>
<swim-progress-spinner mode="determinate" value="100" total="100" appearance="icon" aria-label="Complete"></swim-progress-spinner>
<swim-progress-spinner mode="determinate" value="100" total="100" appearance="icon" is-failure aria-label="Failed"></swim-progress-spinner>

<!-- With swim-icon by name (use ngx icon names, e.g. cloud-upload, check, x) -->
<swim-progress-spinner mode="determinate" value="40" total="100" appearance="icon" in-progress-icon-name="cloud-upload"></swim-progress-spinner>
<!-- Or slot a custom icon -->
<swim-progress-spinner mode="determinate" value="60" total="100" appearance="icon">
  <swim-icon slot="in-progress-icon" font-icon="cloud-upload"></swim-icon>
</swim-progress-spinner>
