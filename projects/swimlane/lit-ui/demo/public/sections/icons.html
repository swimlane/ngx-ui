Icon

Font icons and inline SVG, matching lit-ui (see src/app/icons-page). Uses swim-icon; the icon font is self-loaded by the component (no separate icon CSS required).

    square + x
    square-filled + x (red x overlay)
    field-users + circle-filled (badge)
    Icon with accessible name for screen readers

    Import the icon component; the swim-icon font is self-loaded (no separate CSS required): 

    <script type="module">
  import '@swimlane/lit-ui/icon';
</script>

<swim-icon font-icon="plus"></swim-icon>
<swim-icon font-icon='["square-filled", "x :text-red"]'></swim-icon>
<swim-icon font-icon="x" alt="Close"></swim-icon>
