Progress Spinner

Circular progress indicator matching ngx-progress-spinner: determinate or indeterminate mode, optional center icons (upload, success, failure), and optional labels.

Progress Spinner

Default and determinate modes. Use value and total for progress; when value >= total, the spinner shows complete (and optional icon).

Indeterminate (default)
Determinate 50%
Complete (value = total)

With icon appearance

appearance="icon" shows built-in center icons: upload while in progress, thumbs-up on success, thumbs-down on failure.

In progress (icon)
Complete success
Complete failure

Size and stroke

Diameter and stroke-width control size and thickness.

Small (diameter=50)
Default (100)
Large (150)

With label

spinner-label object provides inProgressLabel, completeLabel, and failLabel (set via property in script).

Label text is set via spinnerLabel property in demo (see main.ts).

Progress Spinner usage

Import and use the progress spinner component:

<script type="module">
  import '@swimlane/lit-ui/progress-spinner';
  import '@swimlane/lit-ui/section';
</script>

<swim-progress-spinner mode="indeterminate" aria-label="Loading"></swim-progress-spinner>
<swim-progress-spinner mode="determinate" value="50" total="100" aria-label="Progress"></swim-progress-spinner>
<swim-progress-spinner mode="determinate" value="100" total="100" appearance="icon" aria-label="Complete"></swim-progress-spinner>
<swim-progress-spinner mode="determinate" value="100" total="100" appearance="icon" is-failure aria-label="Failed"></swim-progress-spinner>
