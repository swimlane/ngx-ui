List

List component with configurable columns, header labels, and row status (error/success/warning). Use swim-list with dataSource, headerLabels, and columns set via JavaScript. Supports scroll-based pagination and custom column layout.

Basic list

Headers: Attack Type, Date of Attack, Origin of Attack. Default row status (error) applied to all rows.

<swim-list id="listBasic"></swim-list>
<script>
  const list = document.getElementById('listBasic');
  list.dataSource = [ { type: 'Malware', date: '1/1/2025', origin: 'China' }, ... ];
  list.headerLabels = ['Attack Type', 'Date of Attack', 'Origin of Attack'];
  list.columns = ['type', 'date', 'origin'];
  list.defaultRowStatus = 'error';
</script>

Custom grid columns (3fr 2fr 1fr)

list.columnLayout = '3fr 2fr 1fr';
list.headerLabels = ['Attack Type', 'Date of Attack', 'Origin of Attack'];
list.columns = ['type', 'date', 'origin'];

Fixed height with page-change on scroll

Current page (from scroll): 1

list.height = 400;
list.paginationConfig = { pageSize: 10 };
list.addEventListener('page-change', (e) => console.log('Page:', e.detail));

Start scrolled to page 5

Current page: 5

list.paginationConfig = { index: 5, pageSize: 10 };

Per-row status from data

Each row has a status field (error, success, warning). No default row status needed.

list.dataSource = [
  { type: 'Malware', date: '1/1/2025', origin: 'China', status: 'error' },
  { type: 'DDOS', date: '1/5/2025', origin: 'China', status: 'warning' },
  ...
];

No status column; default row status

Same data without status; defaultRowStatus applies to all rows.

API

swim-list: Set dataSource (array of records), headerLabels (string[]), columns (string[]; use $index for row number). Optional: column-layout (CSS grid-template-columns), height (px), default-row-status (error | success | warning), paginationConfig ({ pageSize, index? }). Fires page-change (detail: number) and scroll (detail: scrollTop).
