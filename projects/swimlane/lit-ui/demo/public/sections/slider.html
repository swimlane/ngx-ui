Slider

Range slider matching ngx-slider: single or multiple thumbs, optional filled track and tick marks, horizontal or vertical. Form-associated; supports min, max, step, and change events.

Default horizontal slider with default min (0), max (100), step (1).

Default
With initial value

Filled track shows the selected range from the start to the thumb(s).

Two thumbs for a range. Value is comma-separated (e.g. "20,80").

Tick marks at step intervals (or tick-step when set).

step=10, show-ticks
min=10, max=200, step=5, tick-step=20

Disabled state.

Vertical orientation (transform-based, same behavior as horizontal).

Value and percent from the change event are shown below.

Value: 25. Percent: 25

Import and use the slider component:

<script type="module">
  import '@swimlane/lit-ui/slider';
</script>

<swim-slider aria-label="Volume"></swim-slider>
<swim-slider filled value="50" aria-label="Level"></swim-slider>
<swim-slider multiple filled value="20,80" min="0" max="100" aria-label="Range"></swim-slider>
<swim-slider show-ticks step="10" value="50" aria-label="Steps"></swim-slider>
<swim-slider orientation="vertical" value="30" aria-label="Vertical"></swim-slider>
<swim-slider>
