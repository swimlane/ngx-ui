Card

Card container with horizontal or vertical layout, optional status, selection, and outline. Use swim-card with swim-card-header, swim-card-footer, swim-card-avatar, and swim-card-placeholder.

Default orientation with header (avatar, title, subtitle) and accent bar.

AB Horizontal Card Title Subtitle or description line 
<swim-card orientation="horizontal">
  <swim-card-header>
    <swim-card-avatar slot="avatar">AB</swim-card-avatar>
    <span slot="title">Horizontal Card Title</span>
    <span slot="subtitle">Subtitle or description line</span>
  </swim-card-header>
</swim-card>

Properties

AttributeTypeDefaultDescription
orientation'horizontal' | 'vertical''horizontal'Card layout direction
status'success' | 'error' | 'disabled'—Status color indicator
selectablebooleanfalseShows a selection checkbox
selectedbooleanfalseWhether the card is selected
disabledbooleanfalseDisables the card
appearance'normal' | 'flat''normal'Visual style (flat removes shadow)
hide-accentbooleanfalseHides the top accent bar
outline-textstring''Text shown in an outline label

Events

EventDetailDescription
selectbooleanFired when the selection checkbox is toggled
outline-click—Fired when the outline text label is clicked
