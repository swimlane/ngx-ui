Buttons

Button component with variants (default, primary, warning, danger, link, bordered), sizes, labels with <swim-icon> (leading, trailing, stacked, icon-only), host styling via --swim-button-* CSS variables, and states (active, in-progress, success, fail, disabled). Supports promise-based loading state.

CDN: import 'https://widgets.swimlane.app/swim-ui/button.js';

The button below is bound to a variable that cycles through states every 2 seconds. Current state: active.

Cycling state 
// Drive state from a JS variable (e.g. set by setInterval)
const states = ['active', 'in-progress', 'success', 'fail'];
let index = 0;
setInterval(() => {
  index = (index + 1) % states.length;
  const state = states[index];
  btn.setAttribute('state', state);
  labelEl.textContent = state;
}, 2000);

Properties

AttributeTypeDefaultDescription
variant'default' | 'primary' | 'warning' | 'danger' | 'link' | 'bordered''default'Visual style variant
size'small' | 'medium' | 'large''medium'Button size
disabledbooleanfalseDisables the button
state'active' | 'in-progress' | 'success' | 'fail''active'Visual state of the button
loading-textstring''When state is in-progress, shown next to the spinner (default slot stays hidden until active)
type'button' | 'submit' | 'reset''button'HTML button type
.timeoutnumberundefinedWhen state is set by a promise: duration (ms) before returning to active (default 3000). Use 0 to disable auto-reset. Static state buttons use timeout="0" so they do not auto-reset.
.promisePromiseundefinedBind a promise to auto-manage in-progress / success / fail states
