Section

Collapsible section with optional title and custom header. Use swim-section with section-title or swim-section-header for custom header content. Supports legacy, outline, light, and minimal appearances.

CDN: import 'https://widgets.swimlane.app/swim-ui/section.js';

Set --swim-section-* on <swim-section> (or an ancestor). They override the token fallbacks for each appearance. --swim-section-header-hover-background fills the entire header row on hover when the section can be toggled (header-toggle or the default chevron). Hover the sample below to compare.

Custom surfaces (hover the header)
Host, header, content, and header hover colors are driven by the style attribute on this element.

Available overrides

Custom property Default role Notes
--swim-section-background Section host (card surface) Falls back to appearance tokens (e.g. --grey-825 for legacy)
--swim-section-header-background Header bar Outline / minimal use transparent fallbacks
--swim-section-header-hover-background Header on hover Full header row when header-toggle or chevron toggle is present
--swim-section-content-background Content panel Legacy default is transparent; appearance="light" sets a token fallback 
<swim-section
  section-title="Themed section"
  header-toggle
  style="
    --swim-section-background: var(--grey-850);
    --swim-section-header-background: var(--grey-800);
    --swim-section-header-hover-background: var(--blue-500);
    --swim-section-content-background: var(--grey-825);
  "
>
  Body content.
</swim-section>

Import the section bundle (and swim-icon if you use the built-in chevron). Use section-title for a simple title, or slot="header" with <swim-section-header> for custom markup. See API Reference for attributes, events, and CSS variables. 

<script type="module">
  import '@swimlane/swim-ui/section';
  import '@swimlane/swim-ui/icon';
</script>

<swim-section section-title="Attack Details">
  Section body content here.
</swim-section>

<swim-section>
  <swim-section-header slot="header">
    <h3>Custom header</h3>
  </swim-section-header>
  Body content.
</swim-section>
<swim-section>
