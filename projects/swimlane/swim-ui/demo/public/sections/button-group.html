Button Group

Group buttons with shared styling, orientation, and variant (contained or text).

All button group color styles: default (grey) and primary (blue).

Default (horizontal)
One Two Three
Primary (horizontal)
One Two Three
Default (vertical)
One Two Three
Primary (vertical)
One Two Three 
<!-- Default (horizontal) -->
<swim-button-group>
  <swim-button type="button">One</swim-button>
  <swim-button type="button">Two</swim-button>
  <swim-button type="button">Three</swim-button>
</swim-button-group>

<!-- Primary (vertical) -->
<swim-button-group button-group-style="primary" orientation="vertical">
  <swim-button type="button">One</swim-button>
  <swim-button type="button">Two</swim-button>
  <swim-button type="button">Three</swim-button>
</swim-button-group>

Import and use the button group component in your application: 

<script type="module">
  import '@swimlane/swim-ui/button-group';
  import '@swimlane/swim-ui/button';
</script>

<swim-button-group button-group-style="primary">
  <swim-button type="button">Save</swim-button>
  <swim-button type="button">Cancel</swim-button>
</swim-button-group>
