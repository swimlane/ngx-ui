Scrollbars

Utility classes for styled scrollbars matching swim-ui. Apply to scrollable elements (with overflow) or to body for global scrollbars.

swim-scroll

The .swim-scroll class sets overflow: auto and styles the scrollbar. Apply it to an element (e.g. a fixed-height container) or to body for global scrollbars.

Element
<div class="swim-scroll scrollbars-demo-box scrollbars-demo-box--default">
  <div class="scrollbars-demo-content">
    <swim-icon font-icon="graph" icon-class="scrollbars-demo-icon"></swim-icon>
  </div>
</div>
Page
<body class="swim-scroll">
  ...
  <div class="scrollbars-demo-box scrollbars-demo-box--default">
    <swim-icon font-icon="graph" icon-class="scrollbars-demo-icon"></swim-icon>
  </div>
  ...
</body>

swim-scroll-overlay

The .swim-scroll-overlay class is applied per element to only show scrollbars on hover. 

<div class="swim-scroll-overlay scrollbars-demo-box">
  <div class="scrollbars-demo-content">
    <swim-icon font-icon="graph" icon-class="scrollbars-demo-icon"></swim-icon>
  </div>
</div>

swim-scroll-muted

The .swim-scroll-muted class is applied per element to show muted scrollbars that increase opacity on hover. 

<div class="swim-scroll-muted scrollbars-demo-box">
  <div class="scrollbars-demo-content">
    <swim-icon font-icon="graph" icon-class="scrollbars-demo-icon"></swim-icon>
  </div>
</div>
