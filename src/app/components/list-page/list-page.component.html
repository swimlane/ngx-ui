ngx-list Inputs

Name Description
@Input() columnLayout: Partial<CSSStyleDeclaration> The layout of the columns using CSS grid. If this input is not provided, each column will take up 1fr of the available space. See Examples tab for more information.
@Input() dataSource: Array<Record<string, unknown>> = [] The list's source of data where each object in the array represents one list row.
@Input() externalSorting: boolean = false When true, the list emits sort intent via onSort and expects the parent to update dataSource. When false, the list sorts rows internally.
@Input() sort: ListSortPropDir | null = null Active sort state with column prop and direction (asc or desc). Set to null when unsorted. Used to reflect header state and reapply sorting when dataSource changes.
@Input() height: number The height of the containing block for the list's rows. If the content exceeds this height, it will become scrollable.
@Input() rowHeight: number The height of each row in the list must be static to support virtual scrolling. Default is 40px.
@Input() paginationConfig: ListPaginationConfig The configuration object for scroll-based pagination. Configuring this input will enable the component to emit page changes based on page size and the user's scroll position. See Examples tab for more information.
@Input() virtualScroll: boolean = false Whether virtual scrolling is enabled.

ngx-list Outputs

Name Description
@Output() onPageChange: EventEmitter<number> Event emitted when the page changes as a result of scrolling. The paginationConfig Input must be provided for this event to emit.
@Output() onScroll: EventEmitter<number> Event emitted when the list is scrolled. The number emitted is the scrollTop value of the scroll container. This output is provided for implementing a custom pagination solution.
@Output() onSort: EventEmitter<ListSortEvent> Event emitted when a sortable header is clicked. The payload includes the next sort state.

ngx-list-header Inputs

Name Description
@Input() sortable: boolean = false Whether the header can be clicked to sort the list.
@Input() prop?: string The row property name used for sorting when this header is active.
@Input() type?: 'text' | 'date' Sort value type for local sorting. When omitted, prop="date" defaults to date sorting. Use type="date" for date strings such as M/D/YYYY.
@Input() comparator?: (a, b, rowA?, rowB?) => number Optional comparator for local sorting. Used by ngx-list when externalSorting is false.
