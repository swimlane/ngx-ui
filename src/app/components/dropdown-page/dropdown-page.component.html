ngx-dropdown Inputs

Name Description
@Input() closeOnClick: boolean = true Whether the component closes after a dropdown option is selected.
@Input() closeOnMouseLeave: boolean Whether the component closes after the mouse leaves the component. See Examples tab.
@Input() closeOnOutsideClick: boolean = true Whether the component closes after a click outside the component is registered.
@Input() forceDownwardOpening: boolean = true Whether the dropdown menu is forced to open below the component.
@Input() open: boolean Whether the component is open.
@Input() showCaret: boolean Whether the caret is shown. The caret connects from the dropdown menu to the dropdown button.

ngx-dropdown Outputs

No component outputs to display.

ngx-dropdown-toggle Inputs

Name Description
@Input() disabled: boolean Whether the component is disabled.
@Input() showEvent: DropdownShowTypes = DropdownShowTypes.Click How the component is opened. See Examples tab.

ngx-dropdown-toggle Outputs

Name Description
@Output() toggle: EventEmitter<Event> Event emitted when the component is opened.
