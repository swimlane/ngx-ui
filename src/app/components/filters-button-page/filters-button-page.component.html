ngx-filter-button Inputs

Name Description
@Input() label: string = 'All Filters' The text label to display inside the filter button.
@Input() hasFilters: boolean = false Whether any filters are currently applied.
@Input() filterCount: number = 0 The number of filters applied; shown in a count badge.
@Input() disabled: boolean = false Disables the filter button and prevents interaction.
@Input() showCount: boolean = true Whether to show the filter count badge when filters are applied.
@Input() iconPosition: 'left' | 'right' = 'left' Controls whether the filter icon appears on the left or right side of the label.
@Input() showIcon: boolean = true Whether to display the filter icon.

ngx-filter-button Outputs

Name Description
@Output() click: EventEmitter<void> Emits when the filter button is clicked.
