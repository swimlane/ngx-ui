ngx-stepper Inputs

Name Description
@Input() active: number = 0 The inititially active step.
@Input() completeIcon: string = 'ngx-icon ngx-check' The icon to display when all steps in the component have been completed.
@Input() large: boolean Whether the component is displayed larger. See Examples tab.
@Input() position: StepperPosition = StepperPosition.Top The position of the component relative to the trackbar.
@Input() progress: boolean Whether the component displays a progress indicator. See Examples tab.
@Input() readonly: boolean = true Whether the component is readonly. Applies basic CSS styling.
@Input() removeHighlight: boolean Whether background highlight styling is displayed for previously completed steps.
@Input() trackBar: boolean = true Whether the trackbar is displayed underneath the current step. See Examples tab.

ngx-stepper Outputs

Name Description
@Output() activeChange: EventEmitter<number> Event emitted when the active ngx-step changes.

ngx-step Inputs

Name Description
@Input() active: number The active component, zero indexed.
@Input() completeIcon: string The icon to display when the component is complete.
@Input() icon: string The icon to display for the component.
@Input() label: string The label for the component.
@Input() step: number The numerical value assigned to each ngx-step component, zero indexed. Used to determine if the ngx-step component is complete.
@Input() total: number The total number of ngx-step components.

ngx-step Outputs

Name Description
@Output() activeChange: EventEmitter<number> Event emitted when the active component changes.
