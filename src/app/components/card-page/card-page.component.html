ngx-card Inputs

Name Description
@Input() appearance: CardAppearance = CardAppearance.Normal The appearance style of the component, either Normal or Flat.
@Input() disabled: boolean Whether the component is disabled.
@Input() error: boolean Whether the component is displayed in an error state. See Examples tab.
@Input() hideAccent: boolean Whether the accent styling displayed at the end of the component is hidden.
@Input() orientation: CardOrientation = CardOrientation.Horizontal The layout of the component, either Horizontal or Vertical.
@Input() outlineText: string Text to display as part of the outline of the component. See Examples tab.
@Input() selectable: boolean Whether the component can be selected.
@Input() selected: boolean Whether the component is currently selected.
@Input() status: CardStatus The status of the component. Used to display a visual indicator to the user. See Examples tab.
@Input() statusTooltip: string The message to display in the tooltip when hovering on the status indicator.

ngx-card Outputs

Name Description
@Output() outlineClick: EventEmitter<void> Event emitted when the outline of the component registers a click event.
@Output() select: EventEmitter<boolean> Event emitted when the component is selected. The selectable Input must be set to true for this event to emit.

Name Description
