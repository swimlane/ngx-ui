Component Inputs

Name Description
@Input() appearance: SpinnerAppearance = SpinnerAppearance.Default The appearance style of the component.
@Input() color: string = '#1483FF' The color of the component.
@Input() completeIcon: TemplateRef<any> The template to display once loading has completed.
@Input() diameter: number = 100 The width and height of the component, in pixels.
@Input() failIcon: TemplateRef<any> The template to display once loading has failed.
@Input() failStatusColor: string = '#FF4514' The color of the component when the mode is set to determinate, and the loading has failed.
@Input() inProgressIcon: TemplateRef<any> The template to display while loading is in progress.
@Input() isFailure: boolean Whether the component is in a failed state.
@Input() mode: ProgressSpinnerMode = ProgressSpinnerMode.Indeterminate The mode for the component, either determinate or indeterminate.
@Input() spinnerLabel: SpinnerLabel The label for the component in various states.
@Input() strokeWidth: number = 3 The width of the loading bar, in pixels.
@Input() total: number = 100 The numerical value for the completion state of the component. Applicable if mode is set to determinate.
@Input() value: number = 0 The numerical value for the current progress of the component. Applicable if mode is set to determinate.
