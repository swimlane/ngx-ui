ngx-multi-dimension-selection Inputs

Name Description
detached = input<boolean>(false) Whether the component is being used without a parent trigger. When set to true, default values are restored for the opacity, position, transition and z-index CSS properties.
emptyPlaceholder = input<string>('No options available') The placeholder displayed when there are no options available.
filterable = input<boolean>(true) Whether the component supports filtering by search term.
filterEmptyPlaceholder = input<string>('No matches...') The message displayed when no options match the provided filter.
multiple = input<boolean>(true) Whether the component allows multiple selections.
selected = input<Array<string>>([]) The array of selected options. Each array item should correspond to the id property of an element in the selectionList input.
selectionList = input<SelectionList>() The data source used to build each dimension of the component. This data source should include the entire nested structure of each selection list and should have the active property set to true. See the Examples tab for more information.

ngx-multi-dimension-selection Outputs

Name Description
onClose = output<void>() Event emitted when a user presses the Escape key.
onSelectedChange = output<SelectionListOption>() Event emitted when an option is selected or deselected.
