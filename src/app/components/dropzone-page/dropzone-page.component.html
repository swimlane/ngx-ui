Component Inputs

Name Description
@Input() acceptedFileFormats: string[] The acceptable formats for file uploads.
@Input() id: string = `input-${id()}` The ID of the component. The 'id()' in the default value is a random, 5-character alphanumeric sequence.
@Input() multiple: boolean Whether the component allows multiple file selections for upload.
@Input() options: FileUploaderOptions The upload options for the component.
@Input() size: DropzoneSize = DropzoneSize.Large The size of the component.
@Input() uploader: FileUploader The uploader instance for the component.

Component Outputs

Name Description
@Output() afterAddingFile: EventEmitter<{ fileItem: FileItem }> Event emitted when a file has been selected for upload.
@Output() beforeUploadItem: EventEmitter<{ fileItem: FileItem }> Event emitted before a file is uploaded.
@Output() errorItem: EventEmitter<{ response: string, status: number, headers: any }> Event emitted when an error occurs during file upload.
@Output() progressAll: EventEmitter<{ progress: number }> Event emitted when the progress of a file upload changes.
@Output() successItem: EventEmitter<{ item: any, response: string, status: number, headers: any }> Event emitted when a file has been successfully uploaded.
