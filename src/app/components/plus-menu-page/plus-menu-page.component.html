Component Inputs

Name Description
@Input() closeOnClickOutside: boolean = true Whether the component should close once a click outside is registered.
@Input() items: PlusMenuItem[] = [] The selectable menu items to display for the component once opened.
@Input() menuColor: string = '#9FCE36' The color of the component.
@Input() menuTitle: string The title for the component.
@Input() position: PlusMenuPosition = PlusMenuPosition.Right The position of the component menu once opened.

Component Outputs

Name Description
@Output() clickItem: EventEmitter<any> Event emitted when a menu item is selected.
@Output() toggleMenu: EventEmitter<boolean> Event emitted when the component menu is toggled opened/closed.
