Component Inputs

Name Description
@Input() daysOfWeek: string[] = ['S', 'M', 'T', 'W', 'T', 'F', 'S'] The days of the week to display on the component.
@Input() defaultView: CalendarView The default view for the component. Will display year, month, or date. Overrides the minView Input. See Examples tab.
@Input() disabled: boolean Whether the component is disabled.
@Input() inputFormats: Array<string | MomentBuiltinFormat> = ['L', 'LT', 'L LT', moment.ISO_8601] The input formats for the component.
@Input() minDate: Date | string The earliest allowable date to display on the component.
@Input() maxDate: Date | string The latest allowable date to display on the component.
@Input() minView: CalendarView The minimum view for the initial state of the component. Will display year, month, or date. See Examples tab.
@Input() timezone: string The timezone for the component. This will affect what date is listed as the current day.
