Component Inputs

Name Description
@Input() disabled: boolean Whether the component is disabled.
@Input() filled: boolean Whether the component has a filled-in style applied, typically from the start of the component to the currently selected value.
@Input() id: string = 'range-1' The ID of the component.
@Input() min: number = 0 The minimum value allowed.
@Input() max: number = 100 The maximum value allowed.
@Input() multiple: boolean Whether the component allows multiple values through a range selection.
@Input() orientation: 'horizontal' | 'vertical' = 'horizontal' Whether the component is displayed horizontally or vertically.
@Input() showTicks: boolean Whether the component displays tick marks between each allowable value.
@Input() step: number = 1 The stepping interval between allowable numbers e.g. a step of 5 only allows values that are multiples of 5.
@Input() tickStep: number The stepping interval between tick marks. Applicable if showTicks is enabled.

Component Outputs

Name Description
@Output() change: EventEmitter<any> Event emitted when the value of the component is changed.
