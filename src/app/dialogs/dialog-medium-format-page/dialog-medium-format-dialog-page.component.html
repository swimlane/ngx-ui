ngx-large-format-dialog-footer will always be at the bottom of the Dialog with the max height of 64px

ngx-large-format-dialog-footer applies some default styles to set the children to the center with a default gap of 0.5rem. The gap can be customized by using --item-gap CSS variable

You can pass in [styleClass] to completely nullify the default styles.
