Table of Contents

Component Options DialogService Methods

The Dialog component is created using the DialogService.

Listed below are the customization options that can be provided to the DialogService when calling the create() method.

See Examples tab for more information.

Component Options

Name Description
class: string The CSS class to add to the root <div> of the component.
closeButton: boolean = true Whether a close button is displayed in the component.
closeOnBlur: boolean = true Whether the component should close when a blur event is registered.
closeOnEscape: boolean = true Whether the component should close when the escape key is pressed.
content: string The content to display below the component title and custom template.
context: any The custom object to provide to the custom template.
cssClass: string The custom CSS class to add to the dialog content section.
dialogTitle: string The title for the component.
format: DialogFormat = DialogFormat.Regular The format for the component, either Regular or Large.
showOverlay: boolean = true Whether an overlay for the component is displayed.
template: TemplateRef<any> Custom template to display below the component title. See Examples tab.
title: string DEPRECATED. The title for the component. Use the dialogTitle Input instead.
visible: boolean = true Whether dialog content is visible.
zIndex: number = 991 The value for the CSS z-index property to be applied to the component.

DialogService Methods

Name Description Parameters
create Creates an instance of the DialogComponent and returns a ComponentRef<T>. options: DialogOptions
destroy Destroys an instance of the DialogComponent. component: ComponentRef<T>
