Component Options AlertService Methods

The AlertComponent is created using the AlertService.

Listed below are the customization options that can be provided to the AlertService when calling the alert(), confirm(), and prompt() methods.

See Examples tab for more information.

Component Options

Name Description
cancelButtonClass: string | string[] The custom CSS class to add to the cancel button.
cancelButtonText: string = 'Cancel' The text to display on the cancel button.
confirmButtonClass: string | string[] The custom CSS class to add to the confirmation button.
confirmButtonText: string = 'Ok' The text to display on the confirmation button.
content: string The content to display in the component. See Examples tab.
cssClass: string The custom CSS class to add to the component.
longPress: boolean Whether closing the component requires a long press. See Examples tab.
title: string The title for the component.

AlertService Methods

Name Description Parameters
alert Creates an instance of the AlertComponent with the type of Alert and returns a Subject<{ type: string, data: any }>. options: DialogOptions
confirm Creates an instance of the AlertComponent with the type of Confirm and returns a Subject<{ type: string, data: any }>. options: DialogOptions
prompt Creates an instance of the AlertComponent with the type of Prompt and returns a Subject<{ type: string, data: any }>. options: DialogOptions
