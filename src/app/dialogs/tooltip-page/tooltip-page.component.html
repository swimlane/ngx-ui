ngx-tooltip Directive Inputs

Name Description
@Input() tooltipAlignment: AlignmentTypes = AlignmentTypes.center The alignment of the component content.
@Input() tooltipCloseOnClickOutside: boolean = true Whether the component closes when a click outside is registered.
@Input() tooltipCloseOnMouseLeave: boolean = true Whether the component closes when a mouseleave event is registered.
@Input() tooltipContext: any The custom object provided to the component to display data dynamically. See Examples tab.
@Input() tooltipCssClass: string The custom CSS class to add to the component.
@Input() tooltipDisabled: boolean Whether the component is disabled.
@Input() tooltipHideTimeout: number = 300 The duration between when the mouse leaves the host element and the component is hidden, in milliseconds.
@Input() tooltipPlacement: PlacementTypes = PlacementTypes.top The placement of the component.
@Input() tooltipShowCaret: boolean = true Whether the caret connecting the component to the host element is shown.
@Input() tooltipShowEvent: ShowTypes = ShowTypes.all The setting for determining when the component is shown. Options include all, focus, or mouseover.
@Input() tooltipShowTimeout: number = 100 The duration between when the mouse hovers on the host element and the component is shown, in milliseconds.
@Input() tooltipSpacing: number = 10 The space between the host element and the component, in pixels. Defaults to 10, causing the tooltip to render 10px away from the host element.
@Input() tooltipTemplate: TemplateRef<any> The custom template to provide to the component. See Examples tab.
@Input() tooltipTitle: string The title for the component.
@Input() tooltipType: StyleTypes = StyleTypes.popover The style type of the component, either popover or tooltip.
