Component Options

The Notification component is created using the NotificationService.

Listed below are the customization options that can be provided to the NotificationService when calling the create() method.

See Examples tab for more information.

Component Options

Name Description
pauseOnHover: boolean = true Whether the component should pause the timeout while the mouse is hovering to prevent auto-closing.
rateLimit: boolean = true Whether the amount of notifications displayed at one time should be limited.
showClose: boolean = true Whether the component should display a close button.
sound: boolean Whether the component should play a sound when opening.
styleType: NotificationStyleType = NotificationStyleType.none The styling of the component. The selectable style types are info, warning, success, error, or none. See Examples tab.
timeout: number = 3000 The duration between when the component is opened and automatically closed, in milliseconds.
type: NotificationType = NotificationType.html The type of the component, either html or native. See Examples tab.
