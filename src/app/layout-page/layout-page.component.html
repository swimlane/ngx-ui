Layout: Flex

ngx-ui's layout system is a set of CSS classes implemeting flex-box with resonable defaults. This system uses three main classes: .ngx-flex--wrap, .ngx-flex--col, and .ngx-flex--row as well as modifiers as demonstrated below. Of these three classes only .ngx-flex--row is needed for simple flex-box layouts. Containers are optional wrappers that contain ngx-flex--rows. Use the toggle below to see each container layout.
  • .ngx-flex--wrap, .ngx-flex--wrap-fluid is width: 100% at all breakpoints
  • .ngx-flex--wrap-fixed sets a max-width at each responsive breakpoint

Simple

By default children of a ngx-flex--row are autosized. Add any number of child elements to a ngx-flex--row. The ngx-flex--col class is optional for auto sized columns.

Responsive

To make the grid responsive, there are five grid breakpoint modfiers --col-*-* (e.g. --col-sm-4), one for each responsive breakpoint: xs, sm, md, lg, and xl. Note that xs is the default and optional.
